NoSQL:

Name Feature Use Case
MongoDB Provides schemas and sophisticated query. BSON like data Microservice architecture, content management system
dynamoDB Quick reads and writes. Serverless, hence predictable performance. Doesn't contain query language. Gaming leaderboard, user sessions
Cassandra High write throughput, high availability IoT sensor data
Redis In-memory cache leading to low latency on frequent reads User session
Memcached A lightweight version of Redis without persistent data N/A
CouchBase Memory-first database, hence low latency Gaming
CouchDb Offline-first, features mapReduce (aggregate) query functionality, MVCC (concurrency control using versioning), HTTP interface. No internet access mobile phones

Graph DB:

Name Feature Use Case
Neo4j Strong query language, single node, ACID compliant, has built-in features like shortest path N/A
TigerGraph Allows billions of nodes, built on Cassandra. N/A
Neptune Supports RDF (triplet) + Property Graph Interoperability of different kinds of sources, as RDF is W3C compliant
